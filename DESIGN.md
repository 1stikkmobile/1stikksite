# Design

## Style

Medical command center meets community outreach route. The site uses a committed graphite and signal-gold identity, with oxygen teal and clinical red used deliberately for health-service cues.

## Color

- Ink: `oklch(18% 0.03 235)`
- Charcoal: `oklch(25% 0.025 230)`
- Paper: `oklch(98% 0.006 110)`
- Gold: `oklch(87% 0.18 100)`
- Teal: `oklch(72% 0.16 185)`
- Red: `oklch(58% 0.22 28)`
- Moss: `oklch(61% 0.16 135)`

## Typography

Use a strong system sans stack with committed size contrast. Display copy stays readable and direct; no decorative serif, no gradient text, no excessive letter spacing.

## Layout

Full-width section bands separate mission, services, proof, training, leadership, booking, and contact. Repeated items use tight cards only where they are useful. Image-heavy sections use stable aspect ratios and masonry-like rhythm rather than identical grids.

## Motion

Motion is vital-sign and routing based: ECG strokes, mobile route paths, notification pulses, subtle media rail movement, and reveal transforms. All animation is disabled or simplified for `prefers-reduced-motion`.

## Mobile Conversion

On phone, the mock kit is the single priority action. Secondary training options are intentionally compressed or deferred so the user does not face multiple equal-weight decisions at once. Training lists should read left-aligned on mobile, duplicate program sections should be reduced, and the floating call/book actions should support the primary kit CTA rather than compete with it.

## Search Intent

The training funnel should explicitly match "mock drug screening kits" intent without reading like keyword stuffing. The page should explain that the kit is shipped by 1 Stikk, tied to portal support, and completed with live observed mock collections, while internal links connect that training path to the broader drug-screening service page.
